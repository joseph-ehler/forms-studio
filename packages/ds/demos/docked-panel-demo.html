üí° About Docked Panel

On desktop (>1024px), panels use a docked sidebar instead of a sheet. This provides quick access to filters/settings without obscuring the main content.

  • Click "Filters" button to toggle panel
  • Background remains fully interactive
  • Max width: min(420px, 35vw)
  • Z-index: 700 (panel lane)
  • Slides in/out smoothly

üì± Mobile Behavior

On mobile (<768px), the same ResponsiveOverlay kind="panel" would render as a SheetPanel (non-modal, gesture-driven).

The DSProvider automatically chooses the right surface based on viewport size and device type.
